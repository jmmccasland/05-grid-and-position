Day one of the Iron Yard was a stressful one. Not because of the firehose of information being shot full blast at my face but because I was maybe taking it too seriously. I was expecting it to be much more uptight and not nearly as welcoming. I felt super prepared from the pre-work but nothing could prepare for the task of combing through my instructors code which was purposefully destroyed for me to fix. Of course I over thought it and made it much more difficult but with some time and a few breaks I was able to figure it out.

Luckily, there was a bit of pattern to it and the only major task was to find the syntax errors in the CSS then match the classes in the CSS to the HTML elements that needed their properties. It would have been impossible without the design prompt though!
